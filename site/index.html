Welcome to scrnabox's documentation!

Scrnabox is a pipeline for scRNA analysis under HPC. This documrentation includes a tutorial for the scRNA-seq data analysis using scrnabox pipeline under an HPC environment (slurm work load manager system).

For an explanation and of how and why to use scrnabox's pipeline, look at tutorial.
